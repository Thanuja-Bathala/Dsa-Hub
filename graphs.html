What is a Graph?

A graph is a non-linear data structure consisting of vertices (nodes) connected by edges. Graphs are used to model relationships between objects such as social networks, maps, and computer networks.

Graph Implementation (JavaScript)

class Graph {
  constructor() {
    this.vertices = new Map();
  }
  addVertex(v) {
    if (!this.vertices.has(v)) this.vertices.set(v, []);
  }
  addEdge(v1, v2) {
    this.addVertex(v1);
    this.addVertex(v2);
    this.vertices.get(v1).push(v2);
    this.vertices.get(v2).push(v1);
  }
}
