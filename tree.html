Trees

What is a Tree?

A tree is a hierarchical data structure consisting of nodes connected by edges. It has a root node at the top and branches that connect to child nodes. Each node can have zero or more children, but only one parent (except the root). Trees are used to represent hierarchical relationships.

Key Characteristics:
  • Hierarchical structure with root at top
  • Nodes connected by edges (no cycles)
  • Each node has one parent (except root)
  • Leaves are nodes with no children
  • Used in file systems, databases, decision trees

Common Tree Types

Binary Tree
Each node has at most 2 children (left and right).
Binary Search Tree
Left child < Parent < Right child (ordered).
Complete Binary Tree
All levels filled except possibly the last.

Real-World Tree Applications

  • File system directories
  • Company organizational charts
  • Abstract syntax trees (compilers)
  • Family trees and genealogy
  • Decision trees in Machine Learning
  • Database indexing (B+ trees)

Binary Tree Node

class TreeNode {
  constructor(data) {
    this.data = data;
    this.left = null;
    this.right = null;
  }
}

class BinaryTree {
  constructor() {
    this.root = null;
  }
  
  // Insert node (level-order)
  insert(data) {
    const newNode = new TreeNode(data);
    
    if (!this.root) {
      this.root = newNode;
      return;
    }
    
    const queue = [this.root];
    
    while (queue.length > 0) {
      const current = queue.shift();
      
      if (!current.left) {
        current.left = newNode;
        return;
      } else if (!current.right) {
        current.right = newNode;
        return;
      } else {
        queue.push(current.left);
        queue.push(current.right);
      }
    }
  }
}
