Traversal Sequence:

Inorder Traversal

  1. Visit left subtree
  2. Visit root node
  3. Visit right subtree

Result: D, B, E, A, F, C, G

Preorder Traversal

  1. Visit root node
  2. Visit left subtree
  3. Visit right subtree

Result: A, B, D, E, C, F, G

Postorder Traversal

  1. Visit left subtree
  2. Visit right subtree
  3. Visit root node

Result: D, E, B, F, G, C, A

Level Order Traversal

  1. Visit nodes level by level
  2. Use queue for implementation

Result: A, B, C, D, E, F, G

Preorder Traversal Implementation

function preorderTraversal(node) {
  if (node === null) return;

  console.log(node.data);        // Visit root
  preorderTraversal(node.left);  // Visit left subtree
  preorderTraversal(node.right); // Visit right subtree
}

Time Complexity

O(n)

Space Complexity

O(h) (h = tree height)
