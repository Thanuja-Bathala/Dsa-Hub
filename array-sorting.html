Array Sorting

Visualize popular sorting algorithms and understand how they rearrange array elements to achieve sorted order.

Bubble Sort

Time (Avg): O(n²)
Time (Best): O(n)
Space: O(1)
Stable: Yes

Compares adjacent elements and swaps if they’re in wrong order.

Selection Sort

Time (All): O(n²)
Space: O(1)
Stable: No
Swaps: Minimal

Finds minimum element and places it at beginning each time.

Insertion Sort

Time (Avg): O(n²)
Time (Best): O(n)
Space: O(1)
Stable: Yes

Builds sorted array one element at a time by insertion.
